Branches

Recapitulatif

 ①  Comment afficher toutes les branches ?

git branch

 ②  Comment ajouter une nouvelle branche ?

git branch  [ nom de la branche]

 ③  Comment changer de branche ?

git checkout  [ nom de la branche]

 ④  Faire une nouvelle branche et s'y rendre ?

git checkout -b  [ nom de la branche]

 ⑤  Supprimer une branche locale ?

git branch -D [ nom de la branche]

 ⑥  Supprimer une branche distante ?

git push [nom_du_depot_distant] :[ nom de la branche]

 ⑦  Propager le changment à tous les ordi?

git fetch  --all  -prune

Merge: Ajouter dans la brancheA le travail fait dans la brancheB

░  git checkout  blancheA

░  git merge  blancheB
