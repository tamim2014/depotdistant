⏰ Evolution CLI
La CLI bin/console est dépassée.
Dans Symfony6 on utilise une nouvelle CLI qui s'appelle symfony. Tout simplement!

Pour ce faire, il faut au préalable, installer un scoop, via le terminal Powershell à l'aide des 2 commandes suivantes:

1)Set-ExecutionPolicy RemoteSigned -scope CurrentUser
2)iwr -useb get.scoop.sh | iex

Ainsi, on peut utiliser ce scoop pour installer la nouvelle CLI

3) scoop install symfony-cli
