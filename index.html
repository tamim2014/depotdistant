Installation Git

Recapitulatif

Etape1: Lancer Git Bash

Etape2: Connecter le dépot locale
   et le dépot distant(init , add , remote add)

  1. Ajouter le dossier .git(dépot local)

    • se positionner sur le projet et saisir la commande suivante:
    • git init

  2. Enregistrer le projet sur le dossier .git

    • git add             .
    • git commit -m "premier commit"

  3. Connecter le dépot local et le dépot distant

    • Ajouter un nouveau dépot distant(github.com)
    • git remote add [origin] [url dépot distant]  ⚠

Etape3: Commencer à bosser

⏩  C'est fini:
  dépôt local ≡ .git , dépôt distant ≡ origin

⏩  Pour envoyer le projet:
  git push [origin] [nom branche distant]

⏩  Pour récupérer un projet:
  git pull [??] [???]
