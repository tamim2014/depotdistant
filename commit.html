Commit

Enregistrer son code ( sur .git) ? Lister les fichiersmodifiés(status) avant de les enregistrer
git status  
git add .

Commiter son code ( sur .git) ? git status  
git add .
git commit -m   "ce que t as fait"

Téléverser sur github ?

git push origin   master

  Si tu est dans la branche: master

Et Si tu est dans la branch B2 ? git checkout   master
git merge  B2
git push origin master

Mise à jour de code ( sans nouveau fichier) ? On remplace l'étape2( git add . ) par l'option -a
git commit -a  -m "blabla"   ou  
git commit -am "blabla"

Lister les commit ?

git   log 

  Pour quitter le log taper: Q

Les changements non commités ?

git   diff

Aller sur un commit ?

git   checkout  SHADuCommit

Revenir à votre branche
  principale(au commit le + récent)

git   checkout  master

Renommer un commit ?

git commit   --amend  -m "nouveau message"

Annuler un commit ?
  (après un push)

git   revert  numero

Annuler les 3 derniers
  commits

git   revert  HEAD~3..HEAD

Annuler les changements
  non commités

git   reset  --hard
