chmod:
• Le premier chiffre fixe l'accès de l'usager propriétaire,
• le deuxième pour groupe propriétaire
• et le dernier pour les « autres ».
Chaque chiffre est la somme d'un ou plusieurs chiffres correspondant à des accès différents:
 4 : lecture
2 : écriture
1 : exécution
0 : aucun

Tous les droits pour le proprietaire et rien pour les autres

chmod 700 nomFichier

Tous les droits pour le proprietaire
"Lecture" et "Execution" pour le reste

chmod 755 nomFichier

"Lecture/Ecriture" pour le proprietaire et son groupe
"Lecture" seulement pour les autres

chmod 664 nomFichier

Deziper le fichier test.zip dans le dossier courant

zip -r test.zip .

Envoyer le fichier test au serveur1

scp test user@sereur1:dossierDest

Ouvrier l'accès de fichier test à tous(all)

chmod -R a+rx test

Même question en utilisant les chiffres

chmod 555 test
